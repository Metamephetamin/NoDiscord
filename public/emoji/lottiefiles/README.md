# LottieFiles Emoji Assets

Animated emoji GIFs downloaded from the free LottieFiles emoji catalog.

Source page: https://lottiefiles.com/free-animations/emoji?type=free
License page: https://lottiefiles.com/page/license

Selection notes:
- Kept only assets whose GIF corners decode as transparent.
- Skipped assets with solid white or otherwise opaque preview backgrounds.
- Per-asset source pages, creator names, and source titles are preserved in `src/utils/lottiefilesEmojiCatalog.json`.
